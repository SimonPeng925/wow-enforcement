# WOW English 电商维权工具包

## 功能概览

| 脚本 | 用途 |
|------|------|
| `jd_extract_v3.py` | 单链接抓取（手动指定一个京东链接） |
| `batch_ipp.py` | **批量处理（推荐）** 读取 Excel 链接列表，自动抓取并生成 IPP 投诉文档 |
| `启动已登录Chrome.bat` | 启动 Chrome 调试模式（复用已登录状态） |
| `批量处理.bat` | 一键启动批量处理工具 |

---

## 快速开始（批量处理）

### 第一步：准备链接 Excel

打开 **`投诉链接模板.xlsx`**，在 B 列「商品链接」填入要投诉的链接：

```
序号 | 商品链接
1    | https://item.jd.com/123456.html
2    | https://item.pdd.com/abc123.html
3    | https://detail.tmall.com/item.htm?id=xxx
```

支持平台：**京东 / 天猫 / 淘宝 / 拼多多**（自动识别）

### 第二步：启动 Chrome 调试模式（推荐）

1. 关闭所有 Chrome 窗口
2. 双击 **`启动已登录Chrome.bat`**
3. 在新窗口中登录京东/天猫等平台
4. 保持浏览器开着

> 如果跳过此步骤，脚本会启动独立浏览器，需要重新登录

### 第三步：运行批量处理

直接双击 **`批量处理.bat`**，或命令行运行：
```
python batch_ipp.py
```

### 第四步：提交 IPP 投诉

处理完成后，打开 `output/batch_日期/` 文件夹：

| 文件 | 用途 |
|------|------|
| `IPP投诉文档_日期.xlsx` | **主要使用** 打开后复制对应列到 IPP 平台表单 |
| `IPP投诉文档_日期.txt` | 投诉理由模板，可直接全选复制 |
| `批量处理汇总_日期.xlsx` | 处理结果总览（侵权商品标红） |

---

## IPP 投诉文档使用说明

### Excel 版（推荐）
1. 打开 `IPP投诉文档_*.xlsx`
2. 按 IPP 平台表单要求，复制对应列的数据
3. 截图证据在对应编号的子文件夹中

### TXT 版
1. 打开 `IPP投诉文档_*.txt`
2. 全选复制所有内容
3. 粘贴到 IPP 平台的「投诉理由」文本框

## 提取的信息字段

每个商品提取：
- **平台** / **商品链接** / **商品标题** / **价格**
- **店铺名称** / **卖家ID** / **销量/评价**
- **截图**（完整页面 + 主图）
- **疑似侵权判定**（自动匹配 WOW English 关键词）

## 常见问题

**Q: 脚本打开新的浏览器而不是用我已经登录的？**
A: 先关闭 Chrome，双击「启动已登录Chrome.bat」后再运行脚本

**Q: 遇到验证码怎么办？**
A: 在浏览器中手动完成验证，脚本会自动等待并继续

**Q: 某件商品提取失败？**
A: 检查该商品链接是否有效；可能是页面加载超时，单独用 `jd_extract_v3.py` 重试

**Q: 拼多多/天猫链接能处理吗？**
A: 可以！`batch_ipp.py` 自动识别平台并使用对应提取器
