# 京东维权提取工具 - 使用说明

## 工具位置
```
wow-enforcement/jd_extract.py
```

## 使用方法

### 方式一：直接运行（上班测试用）
```bash
cd wow-enforcement
python jd_extract.py "京东商品链接"
```

### 方式二：双击快捷方式
```bash
# 编辑 run_jd.bat 内容：
python jd_extract.py %1
```

## 上班测试流程

1. **登录京东**：用你的维权账号登录 https://www.jd.com
2. **打开商品**：在浏览器中找到可疑商品，打开详情页
3. **复制链接**：从地址栏复制商品链接
4. **运行脚本**：在命令行粘贴链接运行

```bash
python jd_extract.py "粘贴的链接"
```

5. **等待完成**：脚本会自动截图+提取+保存
6. **查看结果**：打开 `output/jd_日期_时间/` 文件夹

## 输出文件

| 文件 | 用途 |
|------|------|
| `jd_extraction.xlsx` | 结构化数据，可直接导入维权系统 |
| `jd_report.txt` | 举报模板，复制到京东IPP |
| `screenshot.png` | 页面截图当证据 |
| `main_images/` | 商品主图 |

## 维权账号注意事项

京东维权需要：
1. 品牌方维权账号（京东IPP平台）
2. 相关资质证明
3. 商标注册证/授权书

如果你有京东的维权账号，可以登录：
https://ipp.jd.com

## 常见问题

### Q: 信息提取都是"未提取"
A: 需要先在浏览器登录京东账号。脚本访问时会带上登录cookie。

### Q: 提示"检测到自动化访问"
A: 京东反爬较严。可以尝试：
- 先登录再运行
- 使用真实浏览器（非headless模式）

### Q: 想用非headless模式看过程
A: 修改脚本中的 `headless=True` 改为 `headless=False`
