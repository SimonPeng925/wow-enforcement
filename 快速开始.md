# 快速开始指南

## 工具已搭建完成！

### 文件位置
```
wow-enforcement/
├── extract_product.py    # 主程序
├── run.py                # 一键启动器（推荐）
├── run.bat               # Windows双击启动
├── README.md             # 详细说明
└── output/               # 提取结果保存目录
```

## 使用方法

### 方法一：双击运行（最简单）
1. 双击 `run.bat`
2. 输入商品链接，回车
3. 等待自动提取完成

### 方法二：命令行运行
```bash
cd wow-enforcement
python extract_product.py "商品链接"
```

### 方法三：从剪贴板读取
```bash
# 安装剪贴板支持（可选）
pip install pyperclip

# 复制链接后直接运行
python run.py
```

## 工作流程

```
发现可疑商品
    ↓
手动打开商品详情页（确保登录状态）
    ↓
运行提取脚本（3-5秒）
    ↓
自动生成：
  ├── screenshot.png        # 页面截图
  ├── extraction_data.xlsx  # 结构化数据
  └── report_template.txt  # 举报模板
    ↓
复制模板内容 → 粘贴到平台举报表单
    ↓
上传截图作为证据
```

## 注意事项

1. **需要登录状态**：淘宝/天猫商品需要登录才能提取完整信息
2. **网络问题**：如果页面加载慢，增加等待时间
3. **定期清理**：`output/` 目录会积累文件，建议每月整理一次
